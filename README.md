# webAPI

## Conexão com o banco de dados PostgreSQL

<a>Adicionar a dependência NuGET: </a>
- Npgsql.EntityFrameworkCore.PostgreSQL 
- Microsoft.EntityFrameworkCore

## Camadas da API

<h3>Model: </h3>
<a>Será adicionado a essa camada</a>
- Classes
- Interface das Classes

<h3>InfraData: </h3>
<a>Será adicionado a essa camada</a>
- ConnectionContext (Conexão com banco de dados)
- Repositórios

<h3>Controller: </h3>
<a>Será adicionado a essa camada</a>
- Controles de cada classe

<h3>ViewModel: </h3>
<a>Será adicionado a essa camada</a>
- ClasseViewModel
